# Gestion Des Retards Et Absences
Employee Attendance & Lateness Management System (C# / .NET, WPF, EF Core, SQL Server) / Academic Project – 2025 

Designed and developed a desktop application to manage employee absences and lateness tracking in an organizational context. 

Built a user-friendly interface using WPF and implemented business logic with C# following object-oriented programming principles. 

 Integrated a SQL Server database using Entity Framework Core to handle data persistence and relationships. 

 Developed features to record, update, and track employee attendance, as well as generate structured reports. 

 Applied software design concepts including separation of concerns and structured application architecture.
